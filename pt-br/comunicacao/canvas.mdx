Canvas transforma uma conversa em um espaço de trabalho compartilhado. Em vez de deixar decisões, briefings e planos perdidos no histórico do chat, você cria uma página viva dentro da própria conversa ou canal.

Use canvas para registrar o que precisa continuar claro depois que a conversa avança.

## Onde o canvas aparece

Canvas pode existir em:

- conversas diretas;
- grupos;
- canais.

Cada conversa ou canal pode ter seus próprios canvases, separados do restante do ambiente.

## Criar um canvas

Na aba de canvas da conversa ou canal, use o botão de criação para adicionar um novo documento. O canvas começa com título e área de conteúdo editável.

## Edição e salvamento

O conteúdo é salvo automaticamente enquanto você edita. O indicador no topo mostra se há alterações pendentes, se está salvando ou se tudo já foi salvo.

Você pode alterar o estilo do título entre diferentes níveis de destaque, como título principal ou subtítulos.

## Blocos de conteúdo

O editor permite estruturar informações em blocos, como:

- texto;
- títulos;
- listas;
- seções;
- links;
- conteúdo organizado para consulta posterior.

## Excluir canvas

Quem tem permissão pode excluir um canvas pelo menu de opções. A exclusão remove o documento daquela conversa ou canal.

## Boas práticas

- Use o chat para discutir.
- Use o canvas para consolidar.
- Dê nomes claros aos canvases.
- Atualize o conteúdo quando decisões mudarem.
- Evite duplicar documentos que já vivem em uma página oficial de conhecimento.
